# Alfred-Workflow

开发工程师常用工具箱


## 全局预览
![1](3971549001331.jpg)



* `ntm` 时间戳转换，支持标准时间格式与时间戳自动检测转换，回车复制结果到剪贴板
* `n64d` Base64 解码
* `n64e` Base64 编码
* `nmd5` MD5 生成
* `nrdm` 随机字符串生成，输入长度
* `ncny` 数字转人民币大写
* `nu2c` Unicode 码转中文
* `nu8c` UTF-8 码转中文
* `nc2u` 中文转 Unicode 码
* `nip` IP 地址查询
* `ncp` 无格式粘贴
* `nsum` 简单数列求和
* `nfm` 简单格式化
* `nhp` 列出所有支持的命令
* `ng` 浏览作者博客
